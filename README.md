# Dynamic Procurement Agent

 Project Description
The Dynamic Procurement Agent is a disaster‑response system that automates aid procurement using blockchain smart contracts, instant vendor onboarding, and transparent transaction logging. Triggered by disaster alerts, it connects local micro‑vendors to relief agencies, ensuring faster delivery of food, water, medicine, and transport with guaranteed payments and auditable records.

---

 Problem Statement
Disaster aid is slowed by bureaucracy, corruption, and lack of local vendor access.  
Our agent removes delays with instant, transparent procurement.

---

 Motivation
Speed saves lives — faster procurement means quicker relief.  
Local vendors gain fair access, building trust and resilience.

---

 Solution & Key Features
- **Trigger‑based activation:** Disaster alerts automatically launch procurement workflows.  
- **Smart contracts:** Blockchain‑secured agreements with guaranteed payments.  
- **Instant vendor onboarding:** Micro‑vendors join without paperwork delays.  
- **Transparent transactions:** Every step logged to prevent corruption.  
- **Scalable deployment:** Works across floods, earthquakes, pandemics, and more.  

---

 What Makes It Unique
- Combines **speed + trust**: Immediate aid delivery with blockchain accountability.  
- Empowers **local vendors**: Inclusion of micro‑suppliers often ignored in traditional systems.  
- **Auditable by design**: Reviewer‑proof logs for donors, NGOs, and governments.  

---

 System Architecture
- **Frontend:** React/Gradio dashboards for NGOs, vendors, auditors.  
- **Backend:** Node.js + Express orchestrator.  
- **Database:** MongoDB Atlas (Cluster0, project `book`).  
- **Blockchain Layer:** Smart contracts (Ethereum/Hyperledger).  
- **ML Models:** Disaster detection, vendor ranking, anomaly detection.  
- **APIs:** Disaster feeds, logistics providers, payment gateways.  

---

Workflow / Data Flow
Alert → Assess → Vendor Match → Smart Contract → Dispatch → Verify → Pay → Audit**

---

 Users Affected
- **Communities:** Receive food, water, medicine faster.  
- **Local vendors:** Gain fair access and guaranteed payments.  
- **NGOs/Governments/Donors:** Benefit from transparency and efficiency.
